# 镜中城 | City of Mirrors

> 一座城市被镜墙劈成两半：左城只能说真话但极度冷漠，右城满口甜言蜜语却虚伪至极。你是唯一能说两种话的人。

## 关于

真话不等于善良，善良不必是真话。真正的智慧在于何时选择诚实，何时选择温柔。

- **类型**：叙事冒险 / 视觉小说
- **引擎**：Ren'Py 8.x + 网页版 (HTML5/CSS/JS)
- **主题**：真善美 · 正能量

```
│   ├── styles/            # 左城/右城/中性样式
│   ├── transforms/        # 动画系统

    └── index.html         # 单文件可直接打开
```

## 核心机制

### 语言轮盘
每个关键对话通过"语言轮盘"进行选择：
- ❄ **真话** — 直接、准确、可能伤人
- ✿ **假话** — 温柔、美好、可能失实
- ◇ **沉默** — 不说话，只是陪伴

### 镜像值系统
你的选择会累积为隐藏值，影响：
- UI 色调变化（冷蓝 ↔ 暖金）
- 对齐标签（行者 / 言真者 / 织谎者 / 沉默者 / 渡者）
- 最终结局走向

## 如何运行

### Ren'Py 版本
1. 下载 [Ren'Py 8.x SDK](https://www.renpy.org/latest.html)
2. 将 `game/` 目录放入 Ren'Py 项目目录
3. 启动 Ren'Py → 选择项目 → 运行

### 网页版本
直接在浏览器中打开 `web/index.html` 即可游玩。
