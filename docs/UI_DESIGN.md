# 《镜中城》UI 设计稿

> 版本：v1.0 | 最后更新：2026-06-19
> 引擎：Ren'Py 8.x
> 目标分辨率：1920×1080（主）/ 自适应 1280×720
> 风格关键词：克制、诗意、水墨与光影、镜面反射

---

## 一、视觉风格总纲

### 1.1 设计哲学

**"UI 是空气，不是家具。"**

玩家不应该注意到 UI 的存在——UI 应该像空气一样自然地包裹叙事，而不是像家具一样占据注意力。所有 UI 元素都应该有"呼吸感"：留白大于填充，动效大于装饰，克制大于华丽。

### 1.2 视觉关键词

```
克制 · 沉默 · 光影 · 水墨 · 镜面 · 呼吸感 · 留白 · 诗意
```

### 1.3 对比语言：左城 vs 右城

整个游戏的视觉系统建立在"左右对比"的基础上：

| 维度 | 左城（言真之地） | 右城（言美之地） | 镜墙 / 中性区域 |
|------|----------------|----------------|----------------|
| 几何语言 | 直角、锐角、硬边 | 圆角、弧线、软边 | 镜面反射、对称 |
| 线条 | 粗粝、不规则、像刻痕 | 流畅、丝滑、像笔触 | 虚线、渐隐、半透明 |
| 纹理 | 石材、金属、冰霜 | 丝绸、花瓣、糖霜 | 镜面、光晕、粒子 |
| 动效 | 碎裂、下坠、锐利移动 | 融化、飘浮、柔和移动 | 折射、旋转、缓慢消失 |
| 光影 | 硬光、高对比、深影 | 柔光、低对比、浅影 | 逆光、边缘光、丁达尔效应 |

### 2.2 功能色

```

平衡标记色：#7EC8B0（青绿）—— 解锁"恰好"语言后的专属色
警告色    ：#C44B4B（暗红）—— 城市健康度低、NPC关系危险
成功色    ：#5C8A5C（暗绿）—— 城市健康度高、NPC关系良好
```

### 2.3 镜像值色彩映射

玩家的"镜像值"（真话/假话倾向）会影响整个UI的色调：

| 镜像值状态 | UI整体色调变化 | 具体表现 |
|-----------|---------------|---------|
| 真话倾向 >70% | 冷色偏移 | 对话框边缘变蓝，HUD冷色渐变 |
| 假话倾向 >70% | 暖色偏移 | 对话框边缘变金，HUD暖色渐变 |
| 沉默倾向 >40% | 灰度增加 | UI整体饱和度降低 |
| 平衡状态 | 原色 + 青绿微光 | 最和谐的视觉状态 |

### 5.1 对话框系统

#### 5.1.1 左城对话框

```
设计要点：
- 直角矩形，硬边，无圆角
- 背景：半透明深灰 (#1A1A2E, alpha 0.85)

- 内边距：上 30px，左右 40px，下 20px
- 文字区域：左对齐，无动画（文字直接出现）
- 名字标签：嵌入对话框左上角，冷蓝底色

│  │                                                  │ │
│  │  你不是这里的人。                                  │ │
│  │                                                  │ │
│  │  在左城，沉默是最好的通行证。                        │ │
│  │                                                  │ │

│  └──────────────────────────────────────────────────┘ │

视觉细节：
- 对话框出现时：从底部"升起"，像石板从地面生长
- 文字出现时：无动画，直接显示（符合左城的直接感）
- 对话框消失时：文字先消失，然后框体向下沉没
- 边框会根据当前对话的"真话强度"微微发光
```

```
- 对话框出现时：像花瓣展开，从中心向外"绽放"
- 文字出现时：打字机效果，每个字带淡入，尾部有微光
- 对话框消失时：文字化为光点向上飘散，框体像花瓣合拢
- 对话框边缘有缓慢流动的光晕（像蜜糖的流动）
```

#### 5.1.3 旁白对话框

```
设计要点：
- 无明确边框，文字直接浮在画面上
- 文字颜色：银灰 (#C0C0C0)，带微弱的发光
- 位置：屏幕中央偏上（y: 35%）
- 字体：斜体，行高较大 (1.8)
- 效果：文字从透明渐入，停留后渐出

                    ┌─────────────────────────┐
                    │                         │
                    │   你醒来时，面前是一面镜子。 │
                    │                         │
                    │   镜子里的人，是你。        │
                    │   但你不知道那是谁。        │
                    │                         │
                    └─────────────────────────┘

- 出现：文字逐行淡入，每行间隔 0.3 秒
- 停留：足够阅读时间后，整体渐出
- 不使用点击继续——旁白有自己的节奏
```

#### 5.1.4 守镜人 · 影 的对话框

```
设计要点：
- 半透明，像雾气中的文字
- 背景：无（文字直接浮在画面雾气中）
- 文字颜色：银白 (#E8E8E8)，带镜面反射效果
- 位置：屏幕下方偏中（y: 65%）
- 特殊效果：文字出现时，周围有微弱的镜面光晕

              ┌──────────────────────────────┐
              │                              │
              │  你做了选择。                  │
              │                              │
              │  每一个选择，都会改变            │
              │  这条裂缝的形状。               │
              │                              │
              └──────────────────────────────┘
              ↑ 文字周围有微弱的银色光晕

视觉细节：
- 影的对话没有名字标签——TA的名字不是"说"出来的，而是"浮现"出来的
- 文字出现时有镜面折射的微弱光效
- 当影说出关键信息时，文字会微微"震动"
```

#### 5.2.2 轮盘组件详解

**整体容器**
- 尺寸：900×500
- 位置：屏幕正中央
- 背景：毛玻璃效果 (backdrop-filter: blur(20px))
- 出现动画：从中心缩放 + 淡入（0.4 秒，ease-out）
- 消失动画：向外扩散 + 淡出（0.3 秒，ease-in）

**情境提示区**
- 位置：容器顶部，居中
- 字号：22px
- 颜色：#A0A0A0（浅灰，不抢焦点）
- 作用：描述当前情境，让玩家理解为什么要做出选择

**真话选项（左侧）**
- 尺寸：350×280
- 位置：容器左侧
- 背景：渐变冷蓝 (#3A4A5C → #4A6FA5)
- 边框：2px 冷蓝 (#4A90D9)，直角
- 文字：24px，白色，左对齐
- 图标：左侧有一个冰晶符号 ❄
- 悬停效果：边框发光（冷蓝），背景亮度 +10%
- 选中效果：边框闪烁一次，然后整个按钮向左"碎裂"消失
- 文字内容：显示真话选项的完整文本

**假话选项（右侧）**
- 尺寸：350×280
- 位置：容器右侧
- 背景：渐变暖金 (#D4A574 → #E8C9A0)
- 边框：2px 暖金 (#D4A574)，圆角 (12px)
- 文字：24px，白色，左对齐
- 图标：右侧有一个花瓣符号 ✿
- 悬停效果：边缘光晕扩散（暖金），背景亮度 +10%
- 选中效果：文字化为光点向上飘散，然后按钮"融化"消失
- 文字内容：显示假话选项的完整文本

**沉默选项（底部中央）**
- 尺寸：120×50
- 位置：容器底部正中央
- 背景：半透明灰 (#9B9B9B, alpha 0.3)
- 边框：1px 虚线灰 (#9B9B9B)
- 文字：18px，灰色，居中
- 特殊：**没有明确提示，没有图标，没有悬停高亮**
- 隐约提示：选项区域下方有一个极淡的呼吸光效（alpha 0.1 → 0.25，周期 4 秒）
- 快捷键：按 S 键直接选择沉默
- 选中效果：画面短暂变暗，然后恢复，选项区缓缓关闭

## 六、动画系统设计

### 6.1 转场动画

#### 6.1.1 左城 → 右城（穿越镜墙）

```
时长：1.5 秒
描述：画面像玻璃一样碎裂，碎片向两侧飞散，露出背后的右城景色

[阶段 1: 0-0.3s] 画面出现裂纹，从中央向四周扩散
[阶段 2: 0.3-0.8s] 画面碎裂成无数碎片，碎片带镜面反射
[阶段 3: 0.8-1.2s] 碎片飞散，露出右城的暖色背景
[阶段 4: 1.2-1.5s] 右城场景淡入，碎片化为光点消失
```

#### 6.1.2 右城 → 左城（穿越镜墙）

```
时长：1.5 秒
描述：画面像水面一样融化，涟漪从中央扩散，涟漪过后是左城的景色

[阶段 1: 0-0.3s] 画面中央出现涟漪，像水面被触碰
[阶段 2: 0.3-0.8s] 涟漪扩散到全屏，画面变得模糊流动
[阶段 3: 0.8-1.2s] 流动的画面逐渐凝固，形成左城的冷色背景
[阶段 4: 1.2-1.5s] 左城场景淡入，涟漪残影消失
```

#### 6.1.3 章节过渡

```
时长：3.0 秒
描述：画面渐暗，然后以新的色调渐亮

[阶段 1: 0-1.0s] 当前场景渐暗，同时出现章节标题
[阶段 2: 1.0-2.0s] 黑屏中显示新章节的标题和副标题
[阶段 3: 2.0-3.0s] 新场景渐入，章节标题渐出

标题样式：
  第X章：[章节名]
  Chapter X: [English Name]
```

#### 6.1.4 记忆闪回

```
时长：0.5 秒（进入）+ 内容 + 0.5 秒（退出）
描述：画面边缘出现老胶片效果，中心区域变为暖黄色调

[进入] 画面边缘出现老胶片颗粒和划痕，色调变为暖黄
[内容] 闪回内容播放，带轻微的抖动效果
[退出] 胶片效果消失，色调恢复正常
```

```
├── 左城：大提琴独奏、低频和弦、偶尔的钢琴单音
├── 右城：竖琴、玻璃琴、弦乐轻柔拨奏
└── 镜墙：两个旋律的对位交织

[顶层] 事件音效层
├── UI 交互音：按钮悬停、点击、菜单切换
├── 选择音效：真话（冰裂声）、假话（风铃声）、沉默（无音效）
└── 剧情音效：闪回（老胶片声）、记忆碎片（水晶共鸣）
```

| 模式 | 调整内容 |

- 所有文字大小可在设置中调整（80% / 100% / 120% / 150%）

- 所有 UI 交互支持键盘操作
- 语言轮盘支持数字键快捷选择（1=真话，2=假话，S=沉默）
- 支持自定义按键绑定
- 自动播放模式可调节速度

## 十二、设计原则备忘

1. **UI 是空气，不是家具。** 不抢叙事的注意力。
2. **左右对比是核心视觉语言。** 所有 UI 元素都应该能明确区分"左城"和"右城"。
3. **沉默选项必须"隐藏但可发现"。** 不要给提示，但要让找到它的玩家感到被奖励。
4. **镜像值变化应该是渐进的。** UI 色调变化缓慢，不应该有突变。
5. **动效服务于情感。** 碎裂 = 真话的力量，融化 = 假话的温柔，消隐 = 沉默的力量。
6. **留白大于填充。** 宁可少一个元素，也不要多一个装饰。
7. **所有 UI 元素都必须有进入/退出动画。** 没有突然出现的东西。
8. **音效是 UI 的一半。** 好的 UI 音效比好的 UI 视觉更重要。
