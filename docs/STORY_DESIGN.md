# 《镜中城》剧情设计稿

> 版本：v1.0 | 最后更新：2026-06-19
> 类型：叙事冒险 / 视觉小说
> 引擎：Ren'Py 8.x
> 预计时长：8-12 小时（全收集）

---

## 一、游戏概述

### 1.1 一句话描述

一座被镜墙劈成两半的城市，左城只能说真话但冰冷，右城只能说假话但虚伪——你是唯一能说两种话的人，你要学会的不是分辨真假，而是何时诚实、何时温柔。

### 1.2 核心命题

- 真话不等于善良，善良不一定是真话
- 极端的真实是残忍，极端的温柔是虚伪
- 语言的最高形态是"恰好"——在对的时刻，用对的方式，说对的话

### 1.3 情感基调

克制、沉思、偶尔温暖。不是热血激昂的英雄叙事，而是一杯慢慢变凉的茶——你以为它凉了，喝一口才发现，温度刚好。

### 1.4 目标玩家

- 喜欢《极乐迪斯科》《奥伯拉·丁的回归》《蔚蓝》等叙事驱动游戏的玩家
- 对语言哲学、道德困境有兴趣的玩家
- 愿意在游戏中思考"沉默也是一种选择"的玩家

## 二、哲学基础

### 2.1 语言即权力

在游戏中，语言不是工具，而是武器、药方和钥匙。

| 语言类型 | 权力属性 | 正面效果 | 负面效果 |
|---------|---------|---------|---------|
| 真话 | 切割力 | 揭示真相、建立信任、摧毁虚假 | 伤害情感、冻结关系、制造裂痕 |
| 假话 | 治愈力 | 缓解痛苦、制造希望、维系关系 | 麻痹警觉、掩盖问题、积累崩溃 |
| 沉默 | 存在力 | 给予空间、尊重自由、避免干预 | 被误解为冷漠、错失干预时机 |
| "恰好" | 平衡力 | 既真实又温柔，既诚实又有温度 | 极难掌握，需要理解对方和自己 |

### 2.2 真理的三重悖论

贯穿全剧的三个哲学困境：

**悖论一：真实之痛**
> 如果真相会让一个人崩溃，说出真相是勇气还是残忍？

**悖论二：善意之谎**
> 如果一个谎言让一个人在最后时刻感到幸福，这个谎言是善还是恶？

**悖论三：沉默之重**
> 如果你看到了一切却选择不说，沉默是尊重还是背叛？

## 三、世界设定

### 3.1 镜中城总览

镜中城建在一个巨大的环形山谷中，镜墙沿直径将城市一分为二。

```
                    ┌─────────────────────────────────┐
                    │           镜中城全貌              │
                    │                                  │
                    │     左城（言真之地）               │
                    │     ┌──────────┐                 │
                    │     │  石之区   │                 │
                    │     │  刻痕广场 │                 │
                    │     │  静默花园 │                 │
                    │     └─────┬────┘                 │
                    │           │ 镜墙                  │
                    │     ┌─────┴────┐                 │
                    │     │  绘梦区   │                 │
                    │     │  花语广场 │                 │
                    │     │  永笑花园 │                 │
                    │     └──────────┘                 │
                    │     右城（言美之地）               │
                    │                                  │
                    └─────────────────────────────────┘
```

### 3.2 左城 · 言真之地

**物理规则：** 真话有物理力量。说出的真话会改变物质世界。

- 说"这面墙有裂缝"，墙上就会出现裂缝
- 说"你很孤独"，说话者周围会凝结冰霜
- 说"我爱你"——如果这是真的，空气会发出微弱的光

**城市外观：**
- 建筑：粗粝的石材、棱角分明、高墙窄窗，像被无数次真话切割过的几何体
- 色调：冷灰 #3A4A5C、深蓝 #1A2A3E、铁锈 #6B3A3A
- 植物：稀少，多为针叶和荆棘——因为"你很丑陋"这类真话会杀死花朵
- 光影：强烈的硬光，影子像刀刻在地面上
- 声音：脚步声清晰、回声锐利、几乎没有背景音乐

**社会结构：**
- 人们习惯沉默，因为随便开口就可能伤人
- 社交以"最低限度沟通"为主——只说必要的话
- 有一种职业叫"真话过滤师"，负责在重要场合筛选哪些真话可以说
- 表达情感的方式是刻石头——把想说的话刻在石头上，但不会念出来

**居民心态：**
- 以"诚实"为最高美德
- 认为右城的人都是骗子，不可信任
- 情感淡漠，但一旦建立信任，关系极为牢固
- 内心深处渴望温暖，但不知道如何表达

### 3.3 右城 · 言美之地

**物理规则：** 假话有美化力量。说出的假话会暂时改变现实。

- 说"你是最美的"，对方的伤疤会暂时消失
- 说"一切都会好的"，眼前的废墟会暂时变成花园
- 说"我永远爱你"——如果听者相信，空气会弥漫花香

**城市外观：**
- 建筑：丝绸与纸板搭建的华丽布景，远看金碧辉煌，近看布满裂痕
- 色调：暖金 #FFD700、玫瑰粉 #FF69B4、蜜糖色 #DEB887
- 植物：繁茂但脆弱——因为没有真话提供养分，所有植物都需要持续的赞美才能存活
- 光影：柔和的漫射光，影子模糊如梦境
- 声音：风铃、八音盒、低声的赞美合唱，永远有一种温暖的嗡鸣

**社会结构：**
- 人们永远在微笑、赞美、拥抱
- 社交以"互相赞美"为核心——不说好话等于社交死亡
- 有一种职业叫"甜言建筑师"，专门用赞美来修补建筑的裂缝
- 表达情感的方式是画画——把对方画得比真人更美，这是最高礼节

**居民心态：**
- 以"让人快乐"为最高美德
- 认为左城的人都是怪物，残忍无情
- 情感丰富但肤浅，因为没有人知道彼此的真实想法
- 内心深处渴望被真正理解，但害怕真话带来的痛苦

### 3.4 镜墙

**物理描述：**
- 高度：贯穿天地，看不到顶端
- 材质：某种未知的银白色物质，表面光滑如镜
- 特性：左城的人看镜子里的右城，看到的是扭曲的、可笑的画面；右城的人看镜子里的左城，看到的是恐怖的、黑暗的画面
- 唯一的裂缝：在镜墙正中央，有一道人形大小的裂缝——你是从这里醒来的

**镜墙的秘密：**
镜墙不是天然形成的，而是由城市的建造者（你）在数千年前建造的。建造的初衷是：当时的人们同时拥有真话和假话的能力，但他们用真话互相攻击，用假话互相欺骗。建造者认为，把这两种破坏力分开，城市就能和平。

但分离并没有治愈问题，只是让两种极端各自发展成了系统性的病态。

镜墙上刻着的字："勿言，渡之"——是建造者留给自己的遗言。"勿言"不是"不要说话"，而是"不要只说一种话"。"渡之"不是"把人送过去"，而是"让他们重新学会完整地说话"。

## 四、角色设计

### 4.1 主角：勿言

| 属性 | 描述 |
|------|------|
| 名字 | 勿言（别人给的名字，自己不记得真名） |
| 外观 | 中性形象，灰色长袍，面部特征模糊（玩家可以代入任何性别和身份） |
| 年龄 | 外表约 25-30 岁，实际年龄不明 |
| 身份 | 失忆者，后揭示为镜中城的建造者 |
| 特殊能力 | 唯一能穿越镜墙的人；唯一能同时说真话和假话的人 |
| 性格 | 由玩家塑造——通过选择塑造TA是偏真话、偏假话、偏沉默，还是平衡 |
| 核心动机 | 最初：找回记忆。中期：弥合城市的裂痕。最终：面对自己的过去，做出选择 |
| 过去 | 曾经是城市的建造者，因一次酒后事故导致孩子死亡，极度自责中投河自尽。冥河（镜墙的前身）给了TA一个选择：永远漂流，或成为摆渡人——渡满九十九人，就能渡自己。但TA选择了一条不同的路——把河劈成两半，造了一座城市，试图让真话和假话不再伤人 |

## 五、剧情章节详细设计

---

### 第一章：醒

**时长：** 约 60-90 分钟
**核心目的：** 建立世界观，让玩家体验左城和右城的本质差异
**情感曲线：** 困惑 → 探索 → 第一次触动

#### 5.1.1 开场

```
场景：镜墙裂缝内部。黑暗，微弱的银白色光。

旁白：你醒来时，面前是一面镜子。
旁白：镜子里的人，是你。
旁白：但你不知道那是谁。

[画面渐亮，显示主角坐在裂缝中，四周是破碎的镜面]

旁白：你的手里有一张纸条，上面写着一个字——
[纸条特写]
"勿言"

旁白：你不记得这是你的名字，还是一个命令。
旁白：裂缝的两边，各有微弱的光。
旁白：一边是冷蓝色，一边是暖金色。

[镜头缓慢转向两侧]

旁白：向哪边走？

[第一个选择：向左 / 向右]
[注意：这个选择不影响最终结局，只决定你先去哪座城]
```

#### 5.1.2 左城路线（先向左走）

```
场景：左城外围。灰色的建筑群，远处有高耸的石塔。

[主角走出裂缝，踏上灰色的石板路]

旁白：空气里有一种奇怪的安静。
旁白：不是寂静——而是所有的声音都被削去了圆润的部分，
旁白：只剩下棱角。

[遇到石匠 白岁]

白岁：（正在雕刻一块石头，头也不抬）
白岁：你不是这里的人。

[语言轮盘出现]
真话选项："我不知道自己是哪里的人。"
假话选项："我是从很远的地方来的。"
沉默选项：[不说话，站在原地]

[选择真话]

#### 5.2.1 主线：左城少年与右城女孩

**相遇：**

**核心冲突：**

**玩家选择路径：**

```

→ 砾说"没关系，我等你"
→ 触发后果：左城 +5 健康，右城 +3 健康

路径 B：教瑶说一句真话
→ 主角在右城教瑶练习说真话，过程艰难
→ 瑶说出"我可能真的喜欢你"，嘴唇裂开出血（真话的代价）
→ 砾听到后，第一次流泪
→ 触发后果：右城 -5 健康（真话伤害了右城的规则），但瑶和砾的关系建立

路径 C：什么都不做，让他们自己找到方式
→ 砾和瑶继续隔着镜墙"对话"，用冰霜和花瓣
→ 几天后，他们在镜墙裂缝处找到了一种方式——砾刻了一个石头心，瑶画了一幅画
→ 两者在裂缝处相遇，形成了一个完整的图案
→ 触发后果：镜墙裂缝变大了一点，但变得更美了

→ 砾崩溃，再也不去镜墙
→ 瑶等了三天，三天后在裂缝处留了一幅画——画里是砾的样子，画得比真人还真实

→ 触发后果：解锁隐藏对话，砾说出"也许假话里，也有真实的东西"
```

### 第四章：虚

**情感曲线：** 危机 → 揭露 → 选择的重量

#### 5.4.1 右城崩塌事件

右城的土地建立在谎言之上。三千年来，无数假话堆叠成了这座城市的基础——每一栋建筑都是由"这栋楼很坚固"这类假话支撑的。但假话的力量有衰减期。当一句假话被重复太多次，或者当说出假话的人开始怀疑自己时，假话就会失效。

**崩塌开始：**

```
场景：右城 · 花语广场

[广场中央的一座花塔开始摇晃]

蜜语：（对着花塔大声）你是最坚固的塔！你永远不会倒！

[假话起作用——花塔暂时稳住了]
[但几秒后，花塔表面出现裂纹——假话的力量在衰减]

蜜语：（崩溃）为什么……为什么不管用了？！

[守镜人 影 出现]

影：因为说这句话的人，已经不相信它了。
影：假话的力量，来自听者的信任。
影：当听者开始怀疑，假话就会碎裂。
影：（看着崩溃的右城）
影：这座城市已经撑了三千年。
影：但没有地基的建筑，终有一天会倒。

[右城长老 花言 出现]

花言：（微笑，但这次的微笑有裂缝）
花言：不用担心。我会处理。
花言：（对主角，低声）你。你是那个能说真话的人。
花言：（第一次说出没有微笑的话）
花言：这座城市快要消失了。
花言：（嘴唇裂开，流出血——这是他三十年来第一次说真话）
花言：救救它。
```

#### 5.4.2 核心抉择：如何拯救右城

```
[语言轮盘，但这次是特殊选择——影响整个右城的命运]

→ "右城需要学会说真话，即使它会痛。"
→ 花言：（震惊）你……要摧毁右城的根基？
→ 选择后果：右城开始改革，但过程痛苦，建筑暂时更脆弱，长期更健康

选项 B：加固假话
→ "让我来帮你说更多的假话，修复这座城市。"
→ 花言：（微笑）明智的选择。
→ 选择后果：右城暂时稳定，但根基更脆弱，第四章结尾崩塌更严重

选项 C：寻找第三条路
→ "也许我们可以用一种新的方式来修补——不完全是真话，也不完全是假话。"
→ 花言：（困惑）什么是"新的方式"？
→ 选择后果：解锁"第三种语言"的线索，但需要更长的时间来实现
```

### 第五章：真

**时长：** 约 120 分钟
**核心目的：** 展示左城的危机，探讨"真"的代价
**情感曲线：** 压抑 → 爆发 → 第一次看见真正的美

#### 5.5.1 左城的规则侵蚀

左城的真话规则也在侵蚀最基本的人际关系。

```
场景：左城 · 静默花园

[一个母亲和孩子在花园里]

孩子：（吃了一口母亲做的饭）
孩子：妈妈，这个饭……
孩子：（犹豫，但左城的规则让他说出真话）
孩子：真的很难吃。

[母亲的手冻伤了——真话的物理伤害]

母亲：（忍住疼痛，轻声）
母亲：我知道。
母亲：（看着冻伤的手）
母亲：我……一直在努力。
母亲：但真话告诉我，我确实不是一个好厨师。

[孩子看到母亲的手，想说什么，但害怕再说出真话]

孩子：（沉默，眼泪流下来）

[主角介入]

[语言轮盘]
真话："你做的饭确实不好吃，但你做的爱是最好的。"（这是一句混合了真话和评价的话）
假话："别担心，下次会好的。"（这是在左城说假话——但你有能力说）
沉默：[蹲下来，帮母亲包扎冻伤的手]

[选择真话]
母亲：（愣住，然后轻声笑了一下）
母亲：爱……是最好的吗？
母亲：（看着孩子）也许……你说得对。
母亲：（手上的冰霜开始融化——真话的治愈力量，当它触动了真实的情感）

[选择假话]
母亲：（看着主角，震惊）
母亲：你……你可以说假话？
母亲：（恐惧）你是什么人？
母亲：（但随即，她摸了摸孩子冻伤的手）
母亲：假话……原来可以说得这么温柔。

→ 这是左城居民第一次体验到假话的正面力量
→ 但也引发了更大的恐惧——如果假话是好的，那左城的一切都建立在什么之上？

[选择沉默]
[你蹲下来，帮母亲包扎手]
[孩子也蹲下来，帮母亲吹手]
[三个人在花园里，不说一句话，但画面里有一种安静的温暖]
→ 有时候，最真实的语言，是不说话
```

#### 5.5.3 第五章关键事件：左城的孩子

```

[主角发现一个孩子独自坐在墙角]

孩子：（看到主角，不说话）

[语言轮盘]
真话："你怎么一个人在这里？"
假话："你看起来需要帮助。"

孩子：但我没想到……爸爸的手……
孩子：（举起爸爸冻伤的手的照片）

→ 有些真话，说出来会伤人，但不说出来，会在心里溃烂

[镜墙的碎片化成了光，融入每个人的身体]
[每个人获得了一种新的能力——既不是纯真话，也不是纯假话]
[而是一种——更接近"恰好"的语言]

[效果：最困难触发的结局，但也是最完整的融合]
[两座城市在沉默中理解了彼此]
```

**结局 B 结尾（假话）：**

```
场景：镜墙碎片前

旁白：镜墙碎裂后的第一年。
旁白：两座城市表面上和平了。
旁白：但主角知道——
旁白：和平建立在一句假话之上。
旁白：这句假话，能撑多久？
旁白：也许一年，也许十年，也许一百年。
旁白：但终有一天，假话会碎裂。
旁白：到那时——
旁白：他们会比三千年前更强大吗？
旁白：还是会被同样的恐惧击溃？

[画面：主角坐在镜墙碎片前，看着远方]
[小默的画在旁边，画里的人还在笑]
[风吹过，画微微动了动]

[镜头拉远，城市全景]
[完]
```

**结局 C 结尾（分别说话）：**

```

旁白：镜墙碎裂后的第一年。
旁白：左城和右城的边界开始模糊。

#### 5.7.4 后日谈（彩蛋结局）

[主角站在河边]

[一个小小的身影从远处跑来]

小默：（扑进主角怀里）

小默：我等了你好久！

## 八、叙事风格指南

### 8.1 对话风格

**左城对话特点：**
- 短句，陈述句为主
- 不用感叹号，极少用问号
- 描述精准，不加修饰
- 示例："你的眼睛是灰色的。这是事实。"

**右城对话特点：**
- 长句，感叹句和问句较多
- 大量比喻和修饰
- 情感饱满，但不触及核心
- 示例："你的眼睛像灰色的宝石，藏着整个宇宙的秘密！"

**主角内心独白风格：**
- 简洁、克制、带有诗意
- 不直接说出情感，用意象暗示
- 示例："镜子里的人在笑。但笑的方式，我好像在哪里见过。"

### 8.2 旁白风格

- 第三人称，全知视角
- 克制、冷静，偶尔带有温度
- 句式简短，节奏如诗
- 示例："风停了。不是因为没有风——而是风也学会了沉默。"

### 8.3 禁止事项

- 不使用网络流行语
- 不使用过度文艺的堆砌辞藻
- 不使用说教式的直白表述
- 不使用"正能量"的口号式语言
- 核心情感通过细节和场景传递，而非直接告知

## 十、设计原则备忘

1. **没有"正确选项"。** 每个选择都有代价，每个选择都有价值。
2. **沉默是一种力量。** 游戏应该奖励那些在某些时刻选择不说话的玩家。
3. **后果延迟显现。** 第一章的选择应该在第四、五章才显现后果。
4. **角色不是NPC。** 每个角色都有自己的弧线，主角的选择影响他们，但不决定他们。
5. **不评判玩家。** 游戏不对真话倾向或假话倾向做价值判断——两种路径都有自己的美丽和代价。
6. **情感优先于机制。** 如果一个机制不能产生情感共鸣，就删掉它。
7. **克制。** 永远克制。少说一句，比多说一句更好。
